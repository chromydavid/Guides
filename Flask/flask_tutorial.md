# FLASK Basics

## Creating virtual eviroment (venv)
 - Creates venv directory:
    `python -m venv env`

- Move to venv:
    ``` bash
        source env/bin/activate
    ```

- Exit venv:
    ``` bash
        deactivate
    ``` 

- Install packages (modules/libraries):
    ``` bash
        pip install Flask Flask-Scss Flask-SQLAlchemy
    ``` 
    - Flask: (main Framework)
    - Flask-Scss: (Sassy CSS)
    - SQLAlchemy: Simplifies comunication to DBs (App to DB middleman)
