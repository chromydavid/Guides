# Markdown Basics -- Formatting Guide

Markdown is a lightweight formatting language used for notes,
documentation, and coding.

------------------------------------------------------------------------

## Headings

Use `#` for headings.

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

------------------------------------------------------------------------

## Text Formatting

-   **Bold** → `**bold**`
-   *Italic* → `*italic*`
-   ***Bold + Italic*** → `***bold italic***`
-   `Inline code` → `` `inline code` ``

------------------------------------------------------------------------

## Lists

### Bullet list

-   Item
-   Item
    -   Sub-item

### Numbered list

1.  First
2.  Second
3.  Third

------------------------------------------------------------------------

## Code Blocks

Use triple backticks and specify the language.

``` python
for i in range(3):
    print(i)
```

``` js
console.log("Hello world");
```

Supported languages: `python`, `js`, `bash`, `html`, `css`, `java`, `c`,
`cpp`
